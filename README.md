

## Getting Started

### Prerequisites

- Node.js (v14.0.0 or later)
- npm (v6.0.0 or later)

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/8bitgentleman/japanese-dictionary-of-color-combinations.git
   ```

2. Navigate to the project directory:
   ```
   cd japanese-dictionary-of-color-combinations
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm run dev
   ```

5. Open your browser and visit `http://localhost:3000`
